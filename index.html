Customization

All customization options can be edited in the script tags in the HTML.
Breakpoint: Set the pixel width you want to trigger the mobile navigation to appear (currently set to 650px).
Position: Change the position between 'top' and 'bottom' based on where on the user's screen you want the menu to appear.
Theme: The theme can be changed between 'light' and 'dark'.
